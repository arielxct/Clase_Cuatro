HTML

Se trata de un lenguaje de etiquetas considerado uno de los lenguajes primordiales de la web, esto ya que es el resposable de brindar la estructura a las paguinas web, por lo tanto es este lenguaje el que defini que elmentos conforman la paguina y de que forma estaran agrupados, pese a esto ya que se trata de un lenguaje confines estructurales se debate el si se deberia considerar un lenguaje de programación o no, ya que como tal HTML no cuanta con capacidades para crear funciones o manipular variables, aun así HTML ofrece una gran variedad de recursos para desarrollar paguinas web.

Temario:

Etiquetas de estructura

Son aquellas que se encargan de estrucutrar las diferentes seciones de la paguina web, actuan como contnenedores para otros elmentos HTML.

Etiquetas de texto

Se tratan de todas quellas etiquetas HTML con la función de estructurar la viasualización del texto de la paguina web y se es necesario atribuirles un valor semantico.

Etiquetas de usos espesificos

Se trata de aquellos elmentos que poseen un unico uso en particular, esto no quere decir que sean etiquetas poco utiles, simplemente estan creadas para cumplir con una unica función.

Etiquetas de Listas

Se tratan de todas aquellas etiquetas usadas para estructurar las listas en las paguinas web.

Etiquetas de metadatos

Se trata de todas aquellas etquetas empleadas para definir los metadatos de una paguina web.
