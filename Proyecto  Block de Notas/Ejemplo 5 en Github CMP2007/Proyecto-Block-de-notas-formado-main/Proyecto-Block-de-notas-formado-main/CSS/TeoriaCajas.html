Teoria de las Cajas

En HTML encuanto a la distribución del espacio en pantalla existen dos tipos de etiquetas, las de Bloque y las de Linea, la diferencia entre embos radica en:

Elemento de Bloque

Las de bloque ocupan todo el secmento orizontal de la paguina en la que se enceuntre, es decir horizontalmente los elemtos de bloque siempre ocuparan el 100% del espacio disponible, por lo que todos los demás elementos del docuemnto se ubicaran debajo de este.

Elemento de Linea

Por otro lado los elementos de linea son mucho más simples, ya que unicamante ocupan el espcio necesario horizontal y verticalmente, por lo que al utilizarse barios elementos de este tipo, estos se visualizaran uno alado del otro.

Una particularidad de los elementos de liena es que no se les puede atribuir propiedades de dimenciones como height y width ya que este tipo de etiquetas unicamente utilizaran el aspacio necesario en base a su contenido.
